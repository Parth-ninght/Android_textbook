# 随手记 - 需求说明文档 (PRD)

## 1. 产品概述

**应用名称**: 随手记  
**产品定位**: 一款专注于离线、私密、沉浸的纯单机本地文字记录应用。  
**目标平台**: Web端，可打包并适配移动版设备 (Android)。  
**核心理念**: 无后台、无强制连网、内容完全保存在本地、纯净无广告的阅读与创作体验。

---

## 2. 功能架构设计

应用主要划分为三大核心模块：**记事本 (Notes)**、**日记 (Diary)**、**作家模式 (Writer)**，以及全局的 **应用设置与数据管理**。

### 2.1 记事本模块 (Notes)
支持简单高效的文本与 Markdown 记录，采用文件夹维度管理。

- **分类与管理**:
  - 支持创建不同文件夹归档笔记。
  - 支持多选模式对笔记进行批量操作（如批量移动至文件夹、批量删除）。
- **内容记录**:
  - 提供富文本与 Markdown 支持，包含粗体、重点列表、代办事项等。
  - 支持在查看时直接勾选并持久化代办事项 (`[ ]` / `[x]` 相互切换并留存进度)。
- **位置记忆**:
  - 自动记忆上次停留的文件夹、最后选中的笔记。

### 2.2 日记模块 (Diary)
基于时间线或简单日期的私密日记系统。

- **快速记录**:
  - 轻松创建日记事件，支持附带标题与情感/天气/文本记录。
  - 支持 Markdown 语法。
- **分类与管理**:
  - 允许按自定义分类管理日记卡片。
  - 具备批量选择、删除的能力。
- **位置记忆**:
  - 下次进入时保持最后查看的目录层级。

### 2.3 作家模式模块 (Writer)
专为长篇内容规划、撰写与沉浸式阅读打造的设计。支持分书、分卷、分章机制。

- **层级架构机制**:
  - **单书结构**: 每本书内包含设定、大纲、草稿与正文卷章目录 (章节和分卷)。
  - **卷与章**: 允许创建分卷 (Volume) 管理章节；支持章节和分卷的拖拽排序。
- **创作助手**:
  - 沉浸创作模式，去除杂乱UI仅保留文字。
  - 字数实时统计、创作时长的自动记录，以及简单的打字测速功能。
- **数据导入与导出**:
  - **快速导入**: 支持通过 `.txt` 文本文件导入单章记录；支持上传特定排版格式的长篇小说 `.txt` 自动拆分为分卷与多章节。
  - **小说导出**: 支持单本书导出，附带大纲图文内容生成标准文本格式。
- **阅读功能管理**:
  - 内置阅读器：支持纯净的背景（可自定义夜间模式）、调整字号大小。
  - 提供横向翻页（H-flip）与纵向滚动阅读 (V-scroll) 的切换。
  - **章节记忆机制**: 开启阅读后返回，能够精准记忆所处书籍的最后阅读页码、滚动位置及章节标识。

### 2.4 跨模块的全局设置与数据管理
强调本地数据的完全掌控力。

- **深色与浅色模式**:
  - 跟随系统风格，亦可手动覆盖配置深色护眼模式。
- **全局记忆能力功能**:
  - 拥有记录 "上次停留应用标签页" (Notes/Diary/Writer) 及内部视图的能力。
- **数据本地备份安全机制 (Import/Export)**:
  - 提供一键备份功能：将三模块的所有数据（记事、日记、设定文档、书本正文及关联图片）合成为一个 `.zip` 文件 （内部分为 `backup.json` 和 `images/`）。
  - 支持备份包导入还原：解决跨设备迁移痛点，数据完全在本地加密处理，彻底杜绝云端泄露风险。

## 3. 非功能性需求说明 (NFR)

- **性能要求**:
  - 对于10万字以上的书籍打开或滚动，需保证应用平滑无阻塞。使用长列表虚拟滑动 (或分页) 等底层实现解决超长文本导致的内存溢出。
- **系统适配特性**:
  - 具备手势支持，为移动端设备提供如左滑呼出等手势绑定。
  - 对 iOS / Android WebView 的适配方案保障导出操作不受移动框架 (如 HBuilder WebView 运行时) 引起的阻断。
- **易用性与一致性设计**:
  - **防误触**: 对于章节批量删除、作品删除，系统强制弹出具有视觉阻断性与危险指示的红字复显 Dialog。

## 4. 后期迭代方向 (Future Roadmap)

1. **富文本插图增强**:
   - 对章节/日记正文内支持跨平台的相册选择插入，并将图文合并为标准文件类型封装。
2. **本地加密 / 隐私锁 (App Lock)**:
   - 增加屏幕级密码锁屏与指纹/面部识别功能。
3. **第三方同步功能**:
   - 引入 WebDAV 标准协议。利用该协议可一键同步给个人 NAS、坚果云等地。
